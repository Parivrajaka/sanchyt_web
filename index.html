How Receipt Scanning Works

1

Select

Pick receipt images from gallery or capture with camera. Batch select up to 50 at once.

2

Scan

On-device OCR extracts amount, merchant, date, and category. Blur detection warns about unclear photos.

3

Review

Edit extracted details, check confidence scores, and resolve duplicates before saving.

4

Save

All expenses saved in one batch. See total impact on your budget instantly.

Privacy & Security

✓

Images stay on device — receipt photos are never uploaded. Only extracted text data is stored in the cloud.

✓

On-device processing — OCR runs locally using Google ML Kit. No server-side image processing.

✓

Encrypted storage — sensitive data stored with AES encryption. API keys secured in device Keychain/Keystore.

✓

Biometric lock — protect app access with fingerprint or face recognition.

✓

Delete your data — permanently wipe all your data anytime. GDPR and DPDP compliant.

✓

Export your data — download all your expenses as PDF, CSV, or JSON at any time.
